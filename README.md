## Desafio vaga de Data Scientist Jr utilizando datasets da Olist
Este desafio consistiu em uma análise exploratória e descritiva dos datasets públicos de **e-commerce da Olist**. Tais análises foram feitas utilizando a linguagem **Python** a partir de conjuntos de dados referentes a pedidos do E-Commerce brasileiro entre os anos de 2016 e 2018. 
![](https://blog.olist.com/wp-content/uploads/2015/02/post-olist-g.png)

### **Contexto:**
A Olist conecta empresas de todo o Brasil a canais sem problemas e com um único contrato. Esses comerciantes podem vender seus produtos através da Olist Store e enviá-los diretamente aos clientes usando os parceiros de logística da Olist.
